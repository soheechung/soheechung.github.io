WordLab

Using pre-trained BERT models, we created an open-source research software for calculating surprisals for multilingual A-maze trials. Initially, the goal was to develop a more accurate model for Mandarin maze trials as Mandarin's orthography requires different tokenization. Upon completion, we expanded the code to use other HuggingFace models, such as multilingual BERT and GPT-2.
